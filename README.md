# 分析として使う軸

- 母数と所持金額

優先順位としてはこんなかんじ

1. 母数と所持金額がおおい
2. 母数が多く所持金額が少ない (薄利多売系)
3. 母数がすくなく所持金額がおおい (顧客満足度充実系)
4. 母数がすくなく所持金学もすくない (攻める価値無し)
